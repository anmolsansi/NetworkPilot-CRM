## Evidence requirements

For a row to move to **Complete**, all of the following must be recorded against one immutable
candidate commit:

1. Backend positive, negative, populated-data, and tenant-isolation cases pass without `skip` or
   `xfail`.
2. The listed frontend tests and relevant authenticated Playwright workflow pass.
3. Ruff, ESLint, backend tests, frontend tests, and production builds pass.
4. Alembic upgrade and downgrade validation passes for schema-bearing versions.
5. The environment, commands, commit SHA, and artifact links are added to this report.
