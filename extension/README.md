# Extension

React MV3 Chrome extension for NetworkPilot CRM.

## Development

```bash
npm install
cp .env.example .env
npm run dev
```

## Build

```bash
npm run build
```

## Install (Unpacked)

1. Run `npm run build`
2. Open `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" and select `dist/` folder
